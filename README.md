# 📊 Live Dashboard of Cryptocurrency

A real-time cryptocurrency analytics dashboard that visualizes live market data, trends, and price movements using interactive charts and dynamic updates.

This project provides traders, analysts, and learners with an intuitive interface to monitor crypto market performance in real time.

---

## 🚀 Features

* Live cryptocurrency price tracking
* Real-time market updates
* Interactive charts and visualizations
* Market cap and volume analysis
* Top gainers and losers tracking
* Clean and responsive UI
* Scalable architecture for additional indicators

---

## 🛠️ Tech Stack

**Frontend / Dashboard**

* Streamlit / Flask (depending on your implementation)

**Backend**

* Python

**Libraries**

* Pandas
* NumPy
* Plotly / Matplotlib
* Requests (API integration)

**Data Source**

* Public Cryptocurrency API (e.g., CoinGecko / Binance API)

## ⚙️ Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Adarshthakur-850/Live-Dashboard-of-Cryptocurrency.git
cd Live-Dashboard-of-Cryptocurrency
```

### 2️⃣ Create Virtual Environment

```bash
python -m venv venv
venv\Scripts\activate   # Windows
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the Application

If using Streamlit:

```bash
streamlit run app.py
```

If using Flask:

```bash
python app.py
```
